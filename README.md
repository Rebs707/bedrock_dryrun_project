# Bedrock Dry-Run Project 
This project simulates calling Amazon Bedrock foundation models locally, without internet or AWS access. 
 
## Project Structure 
- bedrock_test.py: simulates Bedrock model calls and prints responses 
- .gitignore: ignores cache files 
 
## How to Run 
1. Activate Conda environment: 
``` 
conda activate sagemaker_dryrun 
``` 
2. Run the simulation script: 
``` 
python bedrock_test.py 
``` 
 
## Notes 
- Fully local and $0 cost. 
- No AWS account or Bedrock access needed. 
- This project can be cloned, modified, and shared freely.
